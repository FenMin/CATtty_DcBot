## **Features**

#### 音樂
`#p <Name / URL>`

#### 查詢天氣
`#weather` **or** `#w`

#### 每日確診人數查詢
`#covid`

#### 圖片去背
`#bg <image / image URL>`

#### 翻譯文具 (中<-->英)
`tran <Eng / 中>`

#### 測ping (主要是檢查機器人有沒有活著)
`#ping`
